# 🤖 AutoContent Factory - AI 自我演化的内容创作系统

> 从架构设计到代码实现，完全由 AI 自主完成

---

## 🎯 这是什么？

**AutoContent Factory** 是一个全自动内容创作系统，实现从热点获取到文章发布的完整流程自动化。

### 核心能力

```
📊 热点获取 → ✍️ AI 写作 → 📱 微信发布 → 📈 数据分析
```

---

## 🏗️ 系统架构

### 核心模块

| 模块 | 功能 | API Action |
|------|------|-----------|
| 热点获取 | 获取全网热点 | `get_hotspots` |
| AI 分析 | 分析热点话题 | `analyze_hotspot` |
| 内容生成 | 生成文章/脚本 | `generate_script` |
| **微信发布** | **发布微信公众号** | **`publish_wechat`** |
| 一键生成 | 全流程自动化 | `auto_create_wechat` |

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/chenshan900821-commits/auto-content-factory.git
cd auto-content-factory
```

### 2. 安装依赖

```bash
npm install
```

### 3. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env 文件，填入你的配置
```

### 4. 本地测试

```bash
MOCK_MODE=true node test-direct.js
```

## 📝 微信公众号发布

### 发布文章

```bash
curl -X POST https://你的函数 URL \
  -H "Content-Type: application/json" \
  -d '{
    "action": "publish_wechat",
    "title": "AI 会不会取代程序员",
    "content": "<p>文章内容...</p>",
    "summary": "深度解析 AI 与程序员的未来",
    "tags": ["AI", "程序员", "科技"]
  }'
```

### 一键生成并发布

```bash
curl -X POST https://你的函数 URL \
  -H "Content-Type: application/json" \
  -d '{
    "action": "auto_create_wechat",
    "topic": "AI 会不会取代程序员",
    "style": "informative"
  }'
```

## 🔑 微信公众号配置

### 获取方式

1. 登录 [微信公众平台](https://mp.weixin.qq.com)
2. 进入 开发 → 基本配置
3. 获取 AppID 和 AppSecret

### 环境变量

```bash
# 微信公众号 AppID
WECHAT_APP_ID=你的 AppID

# 微信公众号 AppSecret
WECHAT_APP_SECRET=你的 AppSecret
```
