# AutoContent Factory - 自动化内容工厂

全自动内容创作系统：从热点获取到视频上传，一键完成。

## 🎯 功能特性

- ✅ 多平台热点获取（微博/抖音/YouTube/Twitter）
- ✅ AI 自动生成口播稿
- ✅ AI 自动生成视频内容
- ✅ 自动上传到 YouTube/TikTok/视频号
- ✅ 定时发布与频道管理
- ✅ 数据分析与优化

## 🏗️ 系统架构

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  热点获取   │───▶│  脚本生成   │───▶│  视频生成   │───▶│  自动上传   │
│  (Crawler)  │    │   (AI)      │    │   (AI)      │    │   (API)     │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```
